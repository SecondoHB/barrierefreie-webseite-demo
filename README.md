## ♿ Barrierefreiheit (WCAG)

Die Website orientiert sich an den WCAG-Prinzipien:

### 🟢 Wahrnehmbar
- sinnvolle `alt`-Texte für Bilder
- ausreichender Farbkontrast
- klare visuelle Hierarchie

### 🟡 Bedienbar
- vollständige Tastatur-Navigation
- sichtbarer Fokuszustand
- logische Tab-Reihenfolge
- „Skip to Content“-Link

### 🔵 Verständlich
- einfache Sprache
- klare Navigation
- konsistente Struktur

### 🟣 Robust
- semantisches HTML (`header`, `nav`, `main`, `footer`)
- ARIA nur wenn notwendig
- kompatibel mit Screenreadern
